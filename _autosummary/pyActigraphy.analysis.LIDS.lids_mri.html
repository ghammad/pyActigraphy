    pyActigraphy.analysis.LIDS.lids_mri¶

    LIDS.lids_mri(lids, params=None)[source]¶

    Munich Rhythmicity Index

    The Munich Rhythmicity Index (MRI) is defined as \(MRI = A \times r\) with \(A\), the cosine fit amplitude and \(r\), the bivariate correlation coefficient (a.k.a. Pearson’r).

    Parameters

    • lids (pandas.Series) – Output data from LIDS transformation.

    • params (lmfit.Parameters, optional) – Parameters for the fit function. If None, self.lids_fit_params is used instead. Default is None.

    Returns

    mri – Munich Rhythmicity Index

    Return type

    numpy.float64
